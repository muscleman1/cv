  • 'br' tag is used to break the line/add space. (line break element)

  • 'em' is emphasis tag, used to convert the text into italic.
  • 'b' is bold tag,used to make text bolded,but use of 'strong' tag is preferred
  • 'ul' is Unordered Listelement,typically rendered as a bulleted list (bullet points,dots)
  • 'ol' is Ordered Listelement,typically rendered as points with numbers(1,2,3....)
  • 'li' is List Itemelement. It is used to represent an item in a list. It must be contained in a parent element: an ordered list ('ol') or an unordered list ('ul')
